# Project Description:

A complete desktop implementation of the popular board game **Amphipolis**.  
The game faithfully follows the official rules and offers a fully functional graphical interface.


## Technologies

- **Java** (JDK 17+)
- **Object-Oriented Programming**
- **JUnit 5** – Unit Testing



## Features

- Accurate implementation of Amphipolis rules
- Complete GUI
- Legal move validation and highlighting
- Unit tests for core logic
- Executable JAR file


## How to Run

1. Download the project or clone it.
2. Place the entire project inside a folder with the exact name: `C:\Project Java\Project_252_B\`
3. Locate the JAR file at:`out/artifacts/Project_252_B_jar/Project_252_B.jar`
4. Open Command Prompt / Terminal and run: `java -jar Project_252_B.jar`
ΝΟΤΕ: The game uses absolute paths for images and sounds. For best results, place the entire project in C:\Project Java\.
